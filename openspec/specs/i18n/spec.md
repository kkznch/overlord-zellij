## Purpose

CLI 出力メッセージの多言語対応（英語/日本語）を定義する。

## Requirements

### Requirement: 対応言語
システムは英語 (en) と日本語 (ja) の2言語をサポートしなければならない (SHALL)。

#### Scenario: 英語メッセージ
- **WHEN** 言語設定が `en` の時
- **THEN** 全 CLI 出力メッセージが英語で表示される

#### Scenario: 日本語メッセージ
- **WHEN** 言語設定が `ja` の時
- **THEN** 全 CLI 出力メッセージが日本語で表示される

### Requirement: デフォルト言語
言語設定が未指定の場合、デフォルトで英語を使用しなければならない (SHALL)。

#### Scenario: 設定ファイルなし
- **WHEN** `config.toml` が存在しない時
- **THEN** 英語でメッセージが表示される

### Requirement: メッセージの網羅性
全メッセージは対応する全言語の翻訳を持たなければならない (SHALL)。

#### Scenario: コンパイル時チェック
- **WHEN** メッセージ定義に言語の翻訳が欠けている時
- **THEN** コンパイルエラーが発生する
